# 🤖 AuthentiText

A PyTorch based deep learning project for detecting whether text is human written or AI generated. The model uses an LSTM architecture with text preprocessing, sequence encoding, hyperparameter tuning, and evaluation experiments.

---

## 📦 Stack

- Python
- PyTorch
- NumPy
- Pandas
- Matplotlib
- KaggleHub

---

## ✨ Quick start
pip install torch torchvision numpy pandas matplotlib kagglehub datasets scikit-learn
